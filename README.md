# Reclamos_Sanitarios
Practica Proyecto Databricks sobre Reclamos de usuarios a entidades Sanitarias

Este proyecto se centra en el análisis y gestión de reclamos realizados por usuarios a entidades sanitarias, utilizando herramientas de Databricks. El enfoque principal está en las tareas realizadas por el ingeniero de datos, incluyendo la ingesta, limpieza, transformación y modelado de los datos de reclamos. Se busca optimizar la calidad y accesibilidad de la información para facilitar futuras etapas de análisis y toma de decisiones.
